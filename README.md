## 安装

在 WorkBuddy 中直接说：

```
帮我安装玄玑命理技能
```

或手动安装：

```bash
# 下载到 WorkBuddy skills 目录
git clone https://github.com/你的用户名/xuanji-fortune.git ~/.workbuddy/skills/xuanji-fortune
```

## 使用

安装后在 WorkBuddy 中直接提问即可自动触发，例如：

- "帮我排八字，2000年1月1日早上8点，福州"
- "分析我的星盘，1995年6月15日下午3点，北京"
- "帮我起一卦问问今年事业运势"
- "帮我全面分析命理和运势"
